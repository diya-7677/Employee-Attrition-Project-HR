# Employee-Attrition-Project-HR
Project Overview

This project simulates a real-world enterprise HR analytics workflow designed to identify employee attrition drivers using a large-scale synthetic workforce dataset.

Instead of relying on small public datasets, the project focuses on replicating realistic business data challenges and building a complete analytics pipeline similar to what organizations use for workforce decision-making.

The objective was to analyze employee behavior patterns, identify retention risks, and generate business insights that explain why employees leave organizations.

Dataset

Synthetic enterprise-style HR dataset created to simulate real organizational workforce data.

15,000 Employee Records
30+ HR Attributes
Business-driven synthetic dataset with realistic workforce relationships
Sample Variables

Employee ID, Department, Salary, Overtime, Job Satisfaction, Performance Rating, Manager Rating, Engagement Score, Promotion History, Bonus Structure, Work Conditions, Attrition Status.

Project Workflow
1. Business Logic Design

Instead of using public datasets, studied real-world HR attrition reports and workplace behavior patterns to identify major attrition drivers.

Key factors selected:

Salary Compensation
Overtime Workload
Job Satisfaction
Manager Effectiveness
Performance Rating
Work Conditions
Promotion Opportunities
2. Synthetic Data Generation

Generated enterprise-level synthetic HR dataset using Python, Pandas, and NumPy.

Created 15,000 employee records
Designed realistic relationships between workforce variables
Built dataset containing 30+ business-focused HR attributes
3. Data Cleaning & Preprocessing

Performed data cleaning while preserving original business logic relationships.

Main tasks:

Removed duplicate employee records
Standardized inconsistent department labels
Handled missing values using statistical imputation techniques

Methods used:

Department-wise Median Salary Imputation
Department-wise Manager Rating Imputation
Median-based Job Satisfaction Imputation
4. Exploratory Data Analysis (EDA)

Performed business-driven analysis by breaking attrition into multiple workforce segments.

Analysis areas:

Compensation Analysis
Satisfaction Analysis
Work Environment Analysis
Performance Analysis
Managerial Impact Analysis

Focused on identifying patterns affecting employee retention and workforce behavior.

5. Business Insights Generation

Generated workforce insights by analyzing relationships between employee behavior and attrition.

Major focus areas:

Burnout caused by overtime workload
Compensation impact on retention
Managerial effectiveness and attrition
Employee engagement and satisfaction trends
Promotion opportunities and retention behavior
Key Insights

Analysis identified major employee attrition drivers:

Overtime and burnout strongly increase attrition risk
Lower job satisfaction predicts employee exits
Higher engagement scores improve retention
Compensation and bonus structures influence retention decisions
Poor managerial effectiveness increases attrition risk
Promotion opportunities improve long-term retention
